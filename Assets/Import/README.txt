Elevator Game - PS1 Floors (v1)
Scale: meters. Each floor slice is 4m (W) x 3m (D) x 3.65m (H).
Doorway: centered at x=0 on the z=0 plane; 1.6m wide x 2.2m tall.
Place your elevator so its door plane sits at z=0 and aligns to this opening.

Files:
- floor1_lobby/floor1_lobby.obj
- floor2_maintenance/floor2_maintenance.obj
- floor3_office/floor3_office.obj
- floor4_forbidden/floor4_forbidden.obj

All textures are 128x128, quantized ("PS1-style").
Props are separate objects inside each OBJ so you can toggle/animate them independently.
